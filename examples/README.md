# Examples

Synthetic data for demonstrating the analysis pipeline.

These files are NOT real election data. Figures are illustrative only,
modelled loosely on publicly known constituency-level results for Greater
Kailash (AC-50, Delhi) to make the pipeline output interpretable, but
booth-level figures are invented.

Do not use these files for any actual electoral analysis. Use real
Form 20 CSVs from `data/processed/` instead.
